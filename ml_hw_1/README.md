# Исследование потенциальной стоимости продажи автомобиля от его технических, и прочих параметров.

---

## Проделанные этапы:
1) Произведен разведочный анализ данных, по нему:
    - Найден ряд признаков значимо влияющих на отпределение целевого (стоимости);
    - Произведена предобработка данных, признаки с разными единицами измерения приведены к одной.

2) Обучены модели линейной регрессии:
    - Обычная LinearRegression на вещественных признаках;
    - Обычная LinearRegression на стандартизованных вещественных признаках;
    - Модели с L0, L1, L2 регуляризацией на стандартизованных вещественных признаках;
    - Модели с L1, L2 регуляризацией на всех признаках;
    - Модель с L2 резуляризацией на всех признаках с дополнительной генерацией.

Лучшей моделью оказалась последняя, она объясняет 86 процентов распределения целевого признака на тестовой выборке. Она же экспортирована как pickle файл для дальнейшего инференса.

Наибольший буст качества оказывает именно работа с признаками (поиск доплнительных зависимостей, использование полиномиальных признаков и т.д.)

В дальнейшем есть смысл параллельно с исследованием собирать их сразу, чтобы облегчить себе задачу по переиспользованию обученной модели;
- Больше внимания уделить отбору признаков, помогли бы особые параметры catboost, которые позволяют вытаскивать значимость признака при его включении/исключении, или shap графики.
